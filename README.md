# Frontend Projects

This repository contains a collection of **frontend development projects** created as part of my learning journey in web development and computer science.

The projects focus on practicing **HTML, CSS, JavaScript, and modern frontend concepts**, including working with APIs, responsive layouts, and interactive user interfaces.

## Repository Purpose

The goal of this repository is to:

* Practice frontend development skills
* Build small web applications
* Experiment with JavaScript features
* Learn how to work with APIs
* Improve UI and layout design
* Manage projects using **Git and GitHub**

## Technologies Used

The projects in this repository mainly use:

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Fetch API
* Git & GitHub
* Vercel (for deployment)

## Projects Included

Examples of projects that may appear in this repository include:

* Colour changer
* Counter app
* Login form validation
* Remote APIs project
* JavaScript exercises
* Small interactive UI components

Each project usually contains its own folder with the necessary files.

## Learning Focus

The main skills practiced in these projects include:

* DOM manipulation
* Event handling
* Form validation
* Working with remote APIs
* Asynchronous JavaScript (async / await)
* Responsive layouts
* Clean and readable code structure

## Version Control

All projects are tracked using **Git** and pushed to **GitHub** with meaningful commits to document the development process.

## Deployment

Some projects are deployed using **Vercel** so they can be accessed and tested online.
